Dokumentasi API

Base URL: https://your-domain.vercel.app/api

GET/api/snippets
Ambil semua snippet ▼
Endpoint publik — tidak butuh auth. Returns array snippet tanpa snippet_key_hash.
Contoh Request
curl
curl https://your-domain.vercel.app/api/snippets
200 OK · application/json
[{ "id":1, "author":"manzxy", "title":"Fetch Retry", "language":"JavaScript", "likes":12, "views":87 }]
POST/api/snippets
Like / View counter ▼
Body Parameters
actionreqstring"like" | "unlike" | "view"
idreqnumberID snippet yang ditarget
javascript
await fetch('/api/snippets',{
  method:'POST',
  headers:{'Content-Type':'application/json'},
  body:JSON.stringify({action:'like',id:1})
});
200 OK
{ "likes": 13 }  // atau "views" jika action='view'
POST/api/snippet-create
Upload snippet baru ▼
snippetKey di-hash SHA-256 sebelum disimpan. Simpan key asli kamu — tidak bisa dipulihkan dari DB.
Body Parameters
authorreqstringNama/username penulis
titlereqstringJudul snippet
descriptionreqstringDeskripsi singkat
languageoptstringJavaScript | TypeScript | Python | PHP | Go
tagsoptstringPisah koma: "fetch, async, api"
codereqstringIsi kode snippet
snippetKeyreqstringKey pribadi 3–7 karakter untuk edit/hapus
201 Created
{ "ok": true }

PYPython Scraper
Fetch + filter + export▼
python
import requests, json

BASE = "https://your-domain.vercel.app"

def get_all():
    r = requests.get(f"{BASE}/api/snippets")
    r.raise_for_status()
    return r.json()

def by_lang(snippets, lang):
    return [s for s in snippets if s["language"] == lang]

def upload(payload):
    r = requests.post(f"{BASE}/api/snippet-create", json=payload)
    return r.json()

if __name__ == "__main__":
    data = get_all()
    print(f"Total: {len(data)}")
    js = by_lang(data, "JavaScript")
    print(f"JS snippets: {len(js)}")
    # export ke file
    with open("snippets.json", "w") as f:
        json.dump(data, f, indent=2, ensure_ascii=False)
JSNode.js Scraper
Fetch + bulk upload▼
javascript (node 18+)
// node scraper.js
const BASE = 'https://your-domain.vercel.app';

async function fetchAll() {
  const r = await fetch(`${BASE}/api/snippets`);
  if (!r.ok) throw new Error(`HTTP ${r.status}`);
  return r.json();
}

async function upload(payload) {
  const r = await fetch(`${BASE}/api/snippet-create`, {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify(payload)
  });
  return r.json();
}

(async () => {
  const snippets = await fetchAll();
  console.log(`Loaded: ${snippets.length}`);
  // top liked
  const top = [...snippets].sort((a,b) => b.likes - a.likes)[0];
  console.log('Top liked:', top?.title);
})();
PHPPHP Scraper
cURL wrapper▼
php
<?php
$BASE = "https://your-domain.vercel.app";

function getSnippets($base) {
    $ch = curl_init("$base/api/snippets");
    curl_setopt_array($ch, [
        CURLOPT_RETURNTRANSFER => true,
        CURLOPT_HTTPHEADER     => ['Accept: application/json']
    ]);
    $body = curl_exec($ch);
    curl_close($ch);
    return json_decode($body, true);
}

$snippets = getSnippets($BASE);
echo "Total: " . count($snippets) . " snippets\n";
echo json_encode($snippets[0], JSON_PRETTY_PRINT);
